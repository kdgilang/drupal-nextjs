# Drupal x Nextjs with ezcontent

This project is about cms drupal headless as a restfull api service & nextjs as a webapp

## Installation
requirements
    - [git](https://git-scm.com/downloads)
    - [Docker](https://www.docker.com/products/docker-desktop)
    - [Lando](https://docs.lando.dev/getting-started/installation.html)
    

clone repo:
`git@github.com:kdgilang/drupal-nextjs.git`
copy env file in webapp directory:
`cp .env.example .env`

run this in main root directory
```sh
run.sh install
```

Yay app ready to explore !!!y
